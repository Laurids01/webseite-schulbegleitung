2. Bereitstellung der Website und Hosting (GitHub Pages)

Unsere Website wird auf den Servern von GitHub Pages gehostet, einem Dienst der GitHub Inc., 88 Colin P. Kelly Jr. St, San Francisco, CA 94107, USA (nachfolgend „GitHub").

Wenn Sie unsere Website aufrufen, erhebt GitHub automatisch Daten, die Ihr Browser an den Server von GitHub übermittelt. Dies sind insbesondere:

  • IP-Adresse des zugreifenden Rechners
  • Datum und Uhrzeit des Abrufs
  • Name und URL der abgerufenen Datei
  • Website, von der aus der Zugriff erfolgt (Referrer-URL)
  • Verwendeter Browser und ggf. das Betriebssystem Ihres Rechners

Die Verarbeitung dieser Daten ist technisch erforderlich, um Ihnen die Website anzuzeigen und die Stabilität und Sicherheit zu gewährleisten. Rechtsgrundlage hierfür ist Art. 6 Abs. 1 lit. f DSGVO (berechtigtes Interesse).

Da GitHub ein US-amerikanisches Unternehmen ist, werden Daten in die USA übertragen. Microsoft (als Muttergesellschaft von GitHub) ist nach dem EU-U.S. Data Privacy Framework zertifiziert, wodurch ein angemessenes Datenschutzniveau gewährleistet ist.

3. SSL- bzw. TLS-Verschlüsselung

Diese Website nutzt aus Sicherheitsgründen und zum Schutz der Übertragung vertraulicher Inhalte eine SSL- bzw. TLS-Verschlüsselung. Eine verschlüsselte Verbindung erkennen Sie daran, dass die Adresszeile des Browsers von „http://" auf „https://" wechselt und an dem Schloss-Symbol in Ihrer Browserzeile.

4. Lokale Einbindung von Schriftarten (Google Fonts)

Wir nutzen auf dieser Website Schriftarten (Google Fonts) zur einheitlichen Darstellung. Um den Datenschutz zu maximieren, haben wir diese Schriftarten lokal auf unserem eigenen Webspace (GitHub-Repository) installiert.

Beim Aufruf unserer Website erfolgt keine Verbindung zu den Servern von Google. Somit werden keine Daten (wie z.B. Ihre IP-Adresse) an Google übertragen.
